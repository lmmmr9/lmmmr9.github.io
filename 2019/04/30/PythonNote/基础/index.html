逻辑运算符

x or y

x为True时，条件直接成立，返回x

x为False时，y为True时，条件仍然成立，返回y

x and y

x为True，始终返回y

x为False，条件永远不成立，返回0

多个模块import时

避免使用
from X import Y

此时修改Y的值，是修改了局部变量Y的指向，源模块中Y没有变化

应使用
import X

此时修改X.Y的值，修改了X.Y的指向，源模块的Y发生了变化
